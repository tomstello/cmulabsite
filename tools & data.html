Tools & Data

Welcome to the Tools & Data page! Here you'll find links to RMarkdown pipelines, GitHub repositories, and Qualtrics templates that support our research. These resources are designed to help you replicate our studies, analyze data, and design your own experiments. Please explore the links below to access these valuable tools.

RMarkdown Pipelines

Our RMarkdown pipelines provide a structured approach to data analysis, ensuring reproducibility and transparency in our research. These pipelines include scripts for data cleaning, preprocessing, statistical analysis, and visualization. Each pipeline is tailored to a specific study and includes detailed documentation to guide you through the process.

GitHub Repositories

Our GitHub repositories host the code and resources for our research projects. You'll find implementations of our models, experimental designs, and data analysis scripts. Each repository includes a README file with instructions on how to set up and run the code. We encourage you to fork these repositories, contribute to the code, and adapt it for your own research.

Qualtrics Templates

We use Qualtrics for designing and conducting our online experiments. Here, you'll find templates for our experimental designs, including surveys, questionnaires, and experimental tasks. These templates are pre-configured with the necessary logic and randomization, making it easy for you to adapt and use them for your own studies.
